# FINEXSA_UI_STYLE_GUIDE_V3 — نظام الستايل الجديد

**الحالة:** Style System V3  
**المصدر:** استعمالات الألوان المرسلة من صاحب المشروع  
**مبدأ التنفيذ:** استبدال الستايل العام مع حفظ ملفات الرجوع.

## الألوان الرسمية واستعمالاتها

- `#1F2933` — Primary: للخلفيات الغامقة والنصوص الأساسية.
- `#FCF6F0` — Secondary: للخلفيات الفاتحة الراقية.
- `#9CA3AF` — Neutral: للعناصر الثانوية والفواصل.
- `#F9F9F7` — Base: للمساحات البيضاء والكروت.
- `#C2A46D` — Accent: للتمييز والخطوط والأيقونات ولمسات الشعار.

## ملفات الرجوع

تم حفظ الستايل السابق داخل:

- `styles.legacy.css`
- `client/css/client.legacy.css`

إذا لم يعجبنا الستايل الجديد، يمكن الرجوع بإعادة تسمية هذه الملفات إلى أسمائها الأصلية.

## ما لم يتم تغييره

لم يتم تغيير Firebase أو Firestore أو Excel Import أو الشركات أو السنوات المالية أو Code_Map أو MIZAN.
